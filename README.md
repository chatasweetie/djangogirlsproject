### How to run Rideminder locally


Create a virtual environment 

```
> virtualenv env
> source env/bin/activate
```

Install the dependencies

```
> pip install -r requirements.txt
```

Run Django server

```
> python manage.py runserver
```


Open your browser and navigate to 

```
http://localhost:8000/
```
